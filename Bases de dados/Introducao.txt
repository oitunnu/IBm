Definições
	▪ Banco de dados: conjunto de dados com uma organização/estrutura específica
	▪ SGBD (sistema gerenciador de base de dados): conjunto de programas para facilitar a manipulação do banco de dados
	▪ Banco de dados + SGBD = sistema de banco de dados (SBD)
		○ SBD: fornece armazenamento eficiente, confiável; é possível colocar parte dos dados em locais diversos; os dados são massivos e persistentes (não somem)

Vantagens
	- Consistência: minimizar incoerências
	- Integração e padronização
	- Independência de dados e operações
	- Sistemas distribuídos
	- Compartilhamento: controle de concorrência; execução de transações
	- Representação de relacionamentos
	- Segurança: back-up; restrições específicas
	- Deduções e ações: regras e gatilhos -> necessita de uma linguagem procedural para programar esses comandos --->> PL/SQL; (SQL é uma linguagem declarativa)
